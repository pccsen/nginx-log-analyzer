# Nginx Log Analyzer (Shell Script)

This project analyzes an Nginx access log and prints:

- Top 5 IP addresses
- Top 5 requested paths
- Top 5 response status codes
- Top 5 user agents

## Project URL
https://roadmap.sh/projects/nginx-log-analyser

## Requirements

- bash
- awk, sort, uniq, head

## Usage

1. Clone the repository: https://github.com/pccsen/nginx-log-analyzer.git
2. Run the script: analyze.sh
